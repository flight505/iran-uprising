# Iran Uprising Digital Memorial

**Digital Divar (دیوار دیجیتال)**

A secure, anonymous memorial and communication platform connecting Iranians inside the country with the global diaspora. This platform honors those killed in the Women, Life, Freedom uprising while providing censorship-resistant communication channels.

## Vision

Like a digital version of the physical walls where Iranians have posted photos of the fallen, this platform transforms an act of mourning into an act of resistance.

**زن، زندگی، آزادی**
*Woman, Life, Freedom*

## Core Principles

1. **Maximum Anonymity** - No user accounts, no tracking, no data retention
2. **Censorship Resistance** - Tor, PWA, offline-first, peer-to-peer sharing
3. **Cultural Authenticity** - Rooted in Iranian mourning traditions
4. **Bridge Not Wall** - Connects those inside Iran with the diaspora
5. **Dignity in Memory** - Every person honored with traditional respect

## Features

- **Memorial Wall (دیوار یادبود)** - Honor the fallen with photos, candles, and flowers
- **Secure Messaging (تخته پیام)** - End-to-end encrypted, anonymous communication
- **Bridge Mode (پل)** - Connect inside/outside Iran, relay volunteering

## Security

- Tor hidden service (.onion) as primary access
- Zero-knowledge architecture
- End-to-end encryption (Signal Protocol)
- Offline-capable PWA

## Tech Stack

- **Frontend:** SvelteKit, Tailwind CSS, Workbox
- **Backend:** Node.js/Fastify, SQLite + SQLCipher
- **Security:** libsignal, Tor integration
